Ember.js Starter App
====================

Prerequisites
-------------

Install **nodejs** >= 0.8.x


### Ubuntu

```bash
sudo add-apt-repository ppa:chris-lea/node.js
sudo apt-get update
sudo apt-get install nodejs
```

### Mac OS X
```bash
brew install node
```

Installation
------------

```bash
npm install -g
bower install
```

Start the server with `grunt server` and prepare for production with `grunt`
